Marcos de trabajo de Java

La palabra inglesa "Framework"(Marco de trabajo) define, en terminos generales, un conjunto estandarisado de conceptos, practicas y criterios para enfocar un tipo de problematica particular que sirve como referencia, para enfrentar y resolver nuevos problemas de indole similar.

En el desarrollo de software, un framework o infraestructura digital es una estructuta conceptual y tecnologica de soporte definido, normalmente con artefactos o modulos de software contretos que puede servir de base para la organizacion y desarrollo de software. Tipicamente hablando puede incluir soporte de programas, bibliotecas, y un lenguaje interpretado, entre otras herramientas, para asi ayudar a desarrollar y unir componentes de un proyecto.
